## 🚀 Installation (30 secondes)

1. Télécharge le dossier de l’extension
2. Va sur `chrome://extensions/`
3. Active le **Mode développeur**
4. Clique sur **« Charger l’extension non empaquetée »**
5. Sélectionne le dossier → Terminé !

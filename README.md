# Claude Peak Monitor

**🪨 Real-time peak hour alerts pour Claude.ai**  
Ne ratez jamais le meilleur moment pour utiliser Claude.ai grâce à des notifications intelligentes et instantanées.

---

## Pourquoi cette extension

Claude Peak Monitor est conçue pour les utilisateurs de Claude.ai qui veulent maximiser leur productivité. Grâce à cette extension :

- Recevez des alertes en temps réel pendant les heures de pointe.
- Optimisez votre temps et vos interactions avec Claude.ai.
- Simplifiez votre workflow et concentrez-vous sur l'essentiel.

---

## Fonctionnalités

| Fonctionnalité                          | Description                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------|
| 🔔 Alertes en temps réel                | Notifications instantanées lorsque les pics de trafic sont détectés.        |
| 📊 Visualisation intuitive              | Affichage clair et simple des heures de pointe.                              |
| ⚡ Optimisation du workflow             | Vous aide à choisir les meilleurs moments pour interagir avec Claude.ai.     |
| 🛠️ Personnalisation                     | Paramètres ajustables selon vos préférences personnelles.                    |
| 🚀 Léger et rapide                       | Extension minimaliste, n’impacte pas la performance de votre navigateur.     |

---

## Installation en 30 secondes

1. Téléchargez l’extension Claude Peak Monitor pour Chrome.  
2. Ouvrez Chrome et allez dans `Menu > Plus d’outils > Extensions`.  
3. Activez le **Mode développeur**.  
4. Cliquez sur **Charger l’extension non empaquetée**.  
5. Sélectionnez le dossier où l’extension a été téléchargée.  
6. Profitez des alertes en temps réel sur Claude.ai !
